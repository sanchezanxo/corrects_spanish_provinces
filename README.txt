== Description ==
This plugin corrects the names of the Spanish provinces that have been incorrectly translated into Spanish, and restores their original toponym.

La Coruña -> A Coruña
Orense -> Ourense
Lérida -> Lleida
Gerona -> Girona

== Installation ==
Simply download and activate it. Nothing else is needed.
